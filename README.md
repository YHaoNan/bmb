# 📔 使用教程
## 1. 基础信息配置
1. 设置页配置个人基础信息
2. 设置页配置个人训练偏好（可选）。*训练偏好是给AI的额外提示，如你可以描述你健身房的固定器械重量跳步为5kg，避免它生成12.5kg这样的数据。*
3. 设置页配置模型（若不配置无法使用AI功能）。建议使用 [deepseek-v4-pro](https://platform.deepseek.com/usage) / [qwen3-max](https://bailian.console.aliyun.com/)（单次调用AI约耗费5000token，deepseek-v4-pro约耗费0.05￥）
4. 设置页面配置悬浮窗权限。配置后将可以启用休息倒计时功能。

## 2. 创建模板
1. 点击首页开始创建，创建你的训练模板
2. 选择文件夹和模板名称
3. 你可以点击“智能生成计划”，输入你的训练意图，AI自动帮你生成计划

> **动作组**：动作组用于定义一项针对特定部位的训练动作，其中可以包含一个主动作和多个平替动作，它们的训练目标是一致的。在训练时，可以根据健身房的设备占用情况自由选择组内任何动作。

## 3. 开始跟练

1. 在运动记录页，或点击模板的开始按钮，都可以直接开始一次运动
2. 你可以通过下面的气泡来快捷选择运动感受，代偿部位等

> - 😋 轻而易举：一般用于热身组，你能够不费任何力气完成组。
> - 😣 有点吃力：一般用于正式组，你能够以标准动作完成组，并有明显对抗感和肌肉紧绷感。
> - 🥵 无法完成：一般用于力竭组或出现在训练的最后，你无法按照预设的次数完成动作。

## 4. AI运动总结

运动完成后，你可以选择进行AI运动总结。

AI将基于本次运动的增肌效率来对运动表现进行评分，分为S/A/B/C/D等级。

AI将逐动作组给出评价并提出建议，在你的能力不足或超出当前模板范围时，可以利用“AI修改计划”按钮来重规划训练模板，AI将基于渐进式超负荷原则来为你重新设计重量、组数，以让你有所提升。

# 👀 其它
